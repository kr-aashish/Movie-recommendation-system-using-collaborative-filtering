### Movie Recommendation System using Collaborative Filtering
#### Aug 2020 - Aug 2020Aug 2020 - Aug 2020
- Used a dataset of over 10000 movie reviews by 610 users for over 1000 movies.
- Reduced the dimensionality of the dataset by filtering out rarely rated movies and rarely rating users.
- Used Python's Surprise library to predict the rating of different movies by different users.
- Finally recommended the predicted top 3 highest rated movie to the user.
